原作者 Hakim El Hattab / 中文化 Poren Chiang

抬起頭來

reveal.js 是個能用 HTML 輕而易舉建立漂亮簡報的架構。
你需要支援 CSS 3D 變形的瀏覽器，才能見識其完整風采。

噢嗨，這是一些備註。在簡報中這些會被隱藏，但你只需開啟講者備忘錄視窗（按下鍵盤上的 's' 鍵）就能看得到。

全域狀態

在投影片上設定 data-state="藍藍路"，"藍藍路" 就會在開啟投影片的時候被加進文件元素的類別（class）裡。這能讓你套用更大規模的樣式變化，例如切換背景。

自訂事件

另外，只要綁定 data-state 所指定的名字，便能以投影片為單位觸發自訂事件。 


Reveal.addEventListener( 'customevent', function() {
	console.log( '觸發了自訂事件' );
} );

投影片背景

在投影片上設定 data-background="#007777"就能改變頁面的背景顏色。支援所有 CSS 色彩格式。

影像背景

<section data-background="image.png">

重複影像背景

<section data-background="image.png" data-background-repeat="repeat" data-background-size="100px">

背景變換

傳給 reveal.js backgroundTransition: 'slide' 這個參數設定，背景就會滑動而非淡出。

覆寫背景變換

你可以用 data-background-transition="slide" 改變單一投影片的背景變換方式。

智慧引句

分為兩種形式，內置（inline）：
「標準的好處在於總是有太多選擇」
以及區塊（block）：

「多年來一直有這麼個理論：千萬隻猴子對著千萬台打字機盲目亂敲，總能重製出一套莎士比亞全集。網際網路卻證明並非如此。」

漂亮的程式碼


function linkify( selector ) {
  if( supports3DTransforms ) {

    var nodes = document.querySelectorAll( selector );

    for( var i = 0, len = nodes.length; i < len; i++ ) {
      var node = nodes[i];

      if( !node.className ) ) {
        node.className += ' roll';
      }
    };
  }
}
